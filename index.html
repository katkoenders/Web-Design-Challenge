Summary: Latitude vs. N

The purpose behind this project is to conduct a series analyses of weather patterns based on weather data that is collected. The data was gathered from the OpenWeatherMap API to create a dataset of over 500 cities. Matplotlib was then used to plot graphs of weather components versus latitude. Weather components included temperature, humidity, cloudiness, and wind speed. The vizualization are displayed on this website along with explanations of trends.
